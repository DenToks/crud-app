# OWED — Debt & Favors Tracker

Track money and favors between friends. Never forget who owes who.

## Features

- Log cash debts and non-monetary favors
- Set due dates with overdue alerts
- Categorize entries (Rent, Food, Transport, etc.)
- Add optional notes to any entry
- People summary with per-person totals
- Filter by Outstanding / Overdue / Settled
- Sort by date, amount, or name
- Search across all entries
- Copy a full summary to clipboard
- Guided first-time tour with spotlight walkthrough
- Installable as a mobile app (PWA)
- All data stored locally in your browser — nothing sent anywhere

## Tech

React + Vite — no backend, no database, no accounts.

## Run locally

```bash
npm install
npm run dev
```
